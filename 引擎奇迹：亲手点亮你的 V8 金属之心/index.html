*注意：在组装前，请找到随附的润滑油。务必在所有需要润滑的地方充分加油，否则机器容易卡顿。
*Note: Before assembly, find the bottled lubricating oil. Be sure to put enough oil in all the places that need to be lubricated, otherwise the machine is easy to get stuck.

步骤 1：活塞连杆组装 Assembly Piston Mechanism

将连杆、活塞销和活塞组装在一起。使用随附的润滑油对活塞销进行润滑。
Assemble the connecting rod, piston pin, and piston. Lubricate the piston pin with the provided lubricating oil.

重复此步骤，为所有 8 个气缸准备活塞连杆组件。
Repeat this step to prepare piston connecting rod assemblies for all 8 cylinders.

步骤 2：曲轴与气缸体连接 Assembly And Connection Between Crankshaft And Cylinder Block

将曲轴放入气缸体，注意方向。用螺丝固定曲轴轴承盖。
Place the crankshaft into the cylinder block, paying attention to the direction. Secure the crankshaft bearing caps with screws.

再次润滑曲轴以确保顺畅运转。
Lubricate the crankshaft again to ensure smooth operation.

步骤 3：连接活塞与曲轴 Connect Pistons And Crankshaft

将活塞连杆组件逐个插入气缸体，并连接到曲轴上。确保活塞可以自由上下移动。
Insert the piston connecting rod assemblies one by one into the cylinder block and connect them to the crankshaft. Ensure the pistons can move freely up and down.

旋转曲轴，检查活塞运动是否协调顺畅。
Rotate the crankshaft to check if the piston movement is coordinated and smooth.

步骤 4：火花塞安装 Spark Plug Installation

将火花塞安装到气缸盖上。
Install the spark plugs into the cylinder head.

步骤 5：气门安装 Valve Installation

安装气门、气门弹簧等组件到气缸盖上。注意零件方向和顺序。
Install the valves, valve springs, and other components onto the cylinder head. Pay attention to the direction and order of the parts.

安装后检查每个气门是否拧紧，高度是否一致。
After the valves are installed, check whether each valve is tightened and whether the height of each valve is at the same height.

步骤 6：凸轮轴座安装 Cam Seat Installation

将凸轮轴座安装到气缸盖顶部。
Install the cam seats onto the top of the cylinder head.

用螺丝固定凸轮轴座。
Secure the cam seats with screws.

步骤 7：凸轮轴安装 Camshaft Installation

安装凸轮轴。注意区分左右侧。
Install the camshafts. Pay attention to distinguish the left and right sides.

在安装凸轮轴前或后进行润滑。
Lubricate the camshafts before or after installation.

用螺丝固定凸轮轴轴承盖。
Secure the camshaft bearing caps with screws.

步骤 8：凸轮同步链轮安装 Installation Of Cam Synchronization Sprocket

找到标有 "RL" (左) 和 "RR" (右) 的链轮，并安装到对应的凸轮轴末端。
Find the sprockets marked "RL" (Left) and "RR" (Right), and install them onto the corresponding camshaft ends.

确认左右链轮上的标记方向正确。
Confirm the direction of the markings on the left and right sprockets.

步骤 9：排气管组装及安装 Exhaust Pipe Assembly and Installation

将排气管装到红色排气歧管底座上，注意方向。
Assemble the exhaust pipes onto the red exhaust manifold base, paying attention to the direction.

将左右排气歧管安装到气缸盖外侧并拧紧螺丝。
Install the left and right exhaust manifolds onto the outer side of the cylinder heads and tighten the screws.

步骤 10：油底壳安装 Installation Of Oil Pan

将油底壳安装到气缸体底部并拧紧螺丝。
Install the oil pan to the bottom of the cylinder block and tighten the screws.

步骤 11：气缸体组装 Cylinder Assembly Installation

将左右气缸体组装到一起，形成 V 形结构。
Assemble the left and right cylinder blocks together to form the V-shaped structure.

步骤 12：曲轴端轴承安装 Crankshaft End Bearing Installation

安装曲轴末端的轴承和固定座。
Install the bearing and retainer at the end of the crankshaft.

步骤 13：启动电机组装及安装 Starter Motor Assembly and Installation

组装启动电机外壳和电机本体，然后安装到引擎本体上。
Assemble the starter motor casing and the motor itself, then install it onto the engine block.

步骤 14：飞轮安装 Flywheel Installation

将蓝色飞轮安装到曲轴末端并拧紧螺丝。
Install the blue flywheel onto the end of the crankshaft and tighten the screw.

步骤 15：发电机组装及安装 Generator Assembly and Installation

组装发电机本体和外壳，安装皮带轮，然后安装到引擎本体上。
Assemble the generator body and casing, install the pulley, then install it onto the engine block.

步骤 16：张紧座组装及安装 Tension Seat Assembly and Installation

组装皮带张紧座。
Assemble the belt tension seats.

将张紧座安装到引擎本体对应位置并拧紧螺丝。
Install the tension seats to the corresponding positions on the engine block and tighten the screws.

步骤 17：正时校准 Corrected Time

在安装正时链之前，确保主链轮（曲轴端）和凸轮轴链轮上的标记对齐，以校准正时。
Before installing the timing chain, ensure the marks on the main sprocket (crankshaft end) and the camshaft sprockets are aligned to set the correct timing.

步骤 18：正时链安装 Timing Chain Installation

安装正时链，连接曲轴链轮和凸轮轴链轮。
Install the timing chains, connecting the crankshaft sprocket and the camshaft sprockets.

步骤 19：启动齿轮安装 Installation The Starter Gear

安装启动齿轮盖。
Install the starter gear cover.

步骤 20：皮带安装 Belt Installation

安装传动皮带，绕过各个皮带轮和张紧轮。
Install the drive belt, routing it around the various pulleys and tensioners.

步骤 21：空气滤清器盖 (进气歧管) 安装 Air Filter Cover (Intake Manifold) Installation

安装黑色的进气歧管。
Install the black intake manifold.

步骤 22：电池和电源盒安装 Battery and Power Box Installation

安装电池和电源盒，连接电线。
Install the battery and power box, and connect the wires.
